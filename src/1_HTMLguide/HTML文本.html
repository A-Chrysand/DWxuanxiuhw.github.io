文本：

段落

  • <p>段落内容</p>

列表

1、有序表

  1. <ol>
  2. <li>单元格内容</li>
  3. <ol>
  4. 有序表会在内容前会出现1、2、3、I、II、III、a、b、c等标号

2、无序表

  1. <ul>
  2. <li>单元格内容</li>
  3. <ul>
  4. 无序表在内容前会出现圆圈/三角形等记号

3、描述列表

  1. <dl>
  2. <dt>单元格内容</dt>
  3. <dd>单元格内容</dd>
  4. <dl>
  5. <dt></dt>内的是表前的自由注释，相当于上文的1、2、3或圆圈
  6. <dd></dd>内的是表中内容，与上文<li></li>的内容一致

若消除列表前面的序号、装饰符，则在css中加入

  • list-style: none;

表格

  1. <table>
  2. <tr>
  3. <td>表格内容</td>
  4. </tr>
  5. <table>
  6. <tr></tr>中表示的是行内容
  7. <td></td>中表示单元格内容

文本格式化

元素 含义 举例
strong 表示内容的重要性
em 表示内容的着重点
mark 表示内容高亮显示 对搜索结果页面或文章中的搜索词进行突出显示
small 表示细则一类的旁注 通常包括免责声明、 注意事项、法律限制、版权信息等
b 希望读者注意的文字 如文档摘要里的关键词、评论中的产品名、文章导语等
i 表示不同于其他文字的文字，具有不同的语态或语气 如分类名称、技术术语、翻译的散文等
sub 定义下标 常见的下标包括化学符号等
sup 定义上标 常见的上标包括商标符号、指数和脚注编号等

连接：

点击[Text]就能跳转到URL

  • <a href="URL">Text</a>

URL可指向：

1、同一页面的某一位置

  1. <a href="./blog.html">博客</a>
  2. <h2 id="toc1">简介</h2>         
  3. 此时点击目录中的简介就会跳转到简介的介绍页面

2、同一网站的不同文件

  • <a href="./blog.html">博客</a>
  • /*可以用来切换到子页面*/

3、指向另一个网站

  • <a href="https://www.baidu.com"title="Baidu搜索">Baidu</a>
  • 跳转到百度

链接属性：

未访问的链接

  • a:link

已访问的链接

  • a:visited

当有鼠标悬停在链接上

  • a:hover

被选择的链接

  • a:active

发送邮件

  • <a href="mailto:someone@email.com">someone</a>
  • 点击上文中的someone就能发送邮件

target属性

  • <a href="URL" target="_blank">HTML</a>
  • 点击URL后在_blank(新建标签页)打开UR
