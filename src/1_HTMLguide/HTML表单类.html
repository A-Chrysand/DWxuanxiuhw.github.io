输入框:

输入普通文本的框框，如账号：

  • <input type="text"/>
样例：文本输入框

输入不给看的文本的框框，如密码：

  • <input type="password"/>
样例：密码输入框

上面的输入框，都可以添加以下属性来提高可读性：

属性名 属性含义
name 输入框的名称(一般用于js控制)
value 输入框的数值类型(如数字型和字符型)
placeholder 在框中加入提示

选择框:

单选框：

  • <input type="radio"/>
样例：选择框： 选项1 选项2

多选框：

  • <input type="checkbox"/>
样例：多选框： 选项1 选项2 选项3 选项4

上面的选择框，都可以添加以下属性来提高可读性：

属性名 属性含义
name 输入框的名称，选择框的名称一定要相同
checked 添加提前勾选选项

下拉选项表：

  1. <select>
  2. <option value="c1"选项1</option>
  3. <option value="c2"选项2</option>
  4. <option value="c3"选项3</option>
  5. </select>
  6. 选项将从上到下依次排序

若需设置默认显示，只需要在<option>里添加selected属性

优化：

在尝试上面的选项后，你发现框框挺难选中的，加以下操作后，点击旁边的文字也能选中框框了，真是夕阳红枪法的福音。

示例代码：

  1. <p><label for="user">用户：</label>
  2. <input type="text" id="user" name="user"/></p>
  3. <p><label for="pwd">密码：</label>
  4. <input type="text" id="pwd" name="password" /></p>

在<input>旁边添加一个<label>，<label for="自定义属性">与<input id="自定义属性">中的两个自定义属性要相等。
